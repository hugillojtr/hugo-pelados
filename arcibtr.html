Paso 1: Elegir un hosting gratuito

Uno de los servicios gratuitos más populares para alojar sitios web estático es GitHub Pages. Aquí te dejo cómo usarlo:

  1. Crea una cuenta en GitHub si aún no tienes una: https://github.com.

  2. Crea un nuevo repositorio:

    • En tu perfil de GitHub, haz clic en el botón "New" para crear un nuevo repositorio.

    • Dale un nombre a tu repositorio (por ejemplo, "hugo-pelados").

    • No es necesario que lo hagas público, aunque es mejor hacerlo si quieres que otros puedan ver tu página.

    • Haz clic en "Create repository".

  3. Luego, habilita GitHub Pages:

    • Dentro del repositorio que acabas de crear, ve a la sección Settings (Configuraciones).

    • Desplázate hacia abajo hasta la sección GitHub Pages.

    • En la sección Source, selecciona main y luego /root para que apunte al directorio raíz.

    • Guarda los cambios.

Paso 2: Subir los archivos de tu página web

Una vez que tienes tu repositorio, debes subir los archivos HTML y las imágenes.

  1. En el repositorio de GitHub, haz clic en "Add file" y selecciona "Upload files".

  2. Selecciona los archivos de tu página web (como los archivos HTML y las imágenes) y cárgalos.

  3. Después de cargar los archivos, haz clic en "Commit changes" para guardarlos.

Paso 3: Ver tu página web

Una vez que hayas subido los archivos, tu página estará disponible en una URL como esta:

php-template
https://<tu-nombre-de-usuario>.github.io/<nombre-del-repositorio>/

Por ejemplo, si tu nombre de usuario es "juan" y el repositorio se llama "hugo-pelados", la URL sería:

arduino
https://juan.github.io/hugo-pelados/

Paso 4: Actualizar tu página web

Cada vez que necesites actualizar tu página, solo tienes que subir los archivos nuevos y GitHub Pages se encargará de mostrar la versión más reciente.

Si prefieres usar otro servicio de hosting gratuito, como Netlify o Vercel, también te puedo guiar en esos pasos. ¡Avísame si necesitas más detalles!
